# What is Git?

## Definition
Git is a **distributed version control system** used to track changes in source code.

## Why Git Exists
- To track code history
- To collaborate with teams
- To recover deleted or broken code

## How Git Works
Git stores changes as **snapshots**, not files.

## Key Features
- Distributed
- Fast
- Secure (SHA-1 hashing)
- Branching support

## Real-Life Example
Google Docs version history = Git for documents

## Summary
Git helps developers track, manage, and collaborate on code efficiently.
