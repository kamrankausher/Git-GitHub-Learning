# Why Git Exists

## The Problem Before Git
Developers used to:
- Create multiple copies of files
- Manually track changes
- Lose work due to mistakes
- Struggle with teamwork

Example:

project_final.py
project_final_v2.py
project_final_last.py


This caused confusion and errors.

---

## Why Git Was Created
Git was created to:
- Track code history efficiently
- Allow safe experimentation
- Support large teams
- Work offline
- Be extremely fast

---

## Key Advantages of Git
- Distributed system
- Snapshot-based storage
- Lightweight branches
- Strong data integrity

---

## Why Git Is Industry Standard
- Used by almost every tech company
- Open-source and free
- Scales to very large projects
- Works with platforms like GitHub

---

## Summary
Git solves the problem of managing code changes in a clean, fast, and reliable way.
